Welcome to Week Three of Programming for Web Applications (Course Code: WDV3322)!

In this course, we will focus on the following key learning objectives:

Master writing effective ES6 while exploring more complex data structures.
Understand and use data types in programming by applying TypeScript.
Develop proficiency in writing SCSS by employing proper nesting techniques, variables, functions, and mixins.
Learn to chain asynchronous JavaScript code effectively using Promises.
Utilize a JavaScript graphics library (PixiJS) to create and manipulate visual canvas assets on the display.
